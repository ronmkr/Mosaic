# Mosaic Home

Mosaic Home is a high-performance, minimalist Chrome extension that transforms your "New Tab" page into a sleek, organized bookmarks dashboard. Designed for productivity and aesthetics, it replaces the standard cluttered search page with a clean grid, custom backgrounds, and an integrated focus clock.

![Icon](icons/icon.svg)

## 🚀 Features

- **Material Grid Layout:** Your bookmarks, organized into a clean, searchable, and responsive grid.
- **Bookmark Management:** Integrated drag-and-drop to move bookmarks into folders.
- **Custom Backgrounds:** Upload any image to personalize your dashboard. Persistent state across browser restarts.
- **Focus Clock:** A minimalist, centered clock and date widget to keep you on track.
- **Native Sync:** Automatically stays in sync with your Chrome bookmarks. No external storage required.
- **Dark Mode Support:** Seemingly switches between light and dark themes based on your OS settings.
- **Secure Internal Navigation:** Native support for `chrome://`, `file://`, and `about:` bookmarks via the extensions API.

## 🛠️ Installation

1. Clone or download this repository.
2. Open Chrome and navigate to `chrome://extensions/`.
3. Enable **Developer mode** (toggle in the top right).
4. Click **Load unpacked** and select the project directory.
5. Open a new tab!

## ⚙️ Tech Stack

- **Vanilla JavaScript (ES Modules):** Modularized for performance and maintainability.
- **Chrome Extensions API (Manifest V3):** High-security, future-proof implementation.
- **Pure CSS:** No bulky frameworks; high-speed rendering with CSS Grid and Backdrop-Filters.
